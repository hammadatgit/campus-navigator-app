Campus Navigator App (Android)
Overview

The Campus Navigator App is an Android-based mobile application designed to help students and visitors navigate the campus efficiently. The application provides an intuitive interface for locating departments, classrooms, offices, and other important facilities within the campus.

The project was developed to demonstrate practical implementation of mobile application development, location-based services, and user-centered interface design.
